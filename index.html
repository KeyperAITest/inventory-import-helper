Inventory Import File Formatter

Upload your inventory CSV file. This tool verifies and normalizes your file so it matches the system’s required import format.

Only StockNumber is required for import. All other fields may be blank. Partial VINs (7+ characters) are supported.

Expected CSV Format Example: 
StockNumber,Make,Model,Year,ExtColor,,IntColor,VIN
A12345,Ford,F-150,2022,Black,,,"1FTFW1E82NFA12345"
